# 证据等级规则

## 目标
用统一的多维评级体系，约束不同资讯在投研中的使用权限，避免弱信息冒充强事实，也避免“真信息”被误判为“高短线价值”。

## 总原则
1. H 只代表证据硬度，不代表短线影响力。
2. L 代表逻辑合理性，判断产业链、业务、收入路径是否顺。
3. C 代表环境适配度，判断是否卡在当下主线与资金偏好上。
4. W 代表证伪窗口，判断短期内是否容易被打假、澄清、证伪或被盘面否定。
5. I 代表影响力预期，判断是否可能影响盘面。
6. 最终评级 SSS / SS / S / A / B / C / R 表示短线博弈影响力，不表示事实真实性。
7. 真实性低不等于影响力低。
8. 真实性高不等于短线影响力高。
9. 同一条资讯可拆成不同部分分别评级。
10. 与更高硬度证据冲突时，低硬度内容必须降级。
11. H 不等于 I。
12. W 不等于 H。
13. W 只代表短期证伪难度。
14. 低 H + 高 W + 高 L + 高 C，可能形成高影响低可信观察。
15. 高 W + 低 L，只能视为空泛吹票，不得升级。
16. W0 一旦出现，最终评级必须降为 R 或进入降级审查。
17. 旧版 E0-E5 已废弃，统一改用 H / L / C / W / I + 最终评级。
18. 时间不改变 H 证据硬度。
19. 时间主要影响 C 环境适配、W 证伪窗口、I 影响力预期和处理结果。

## H：证据硬度

### H5：硬事实
公司公告、监管文件、交易所文件、定期报告、业绩预告、中标公告、订单公告、合同公告、澄清公告。

适用说明：
- 属于正式发布、可回溯、具备法律或监管约束的信息。
- 可直接作为公司事实层面的核心依据。
- 若市场传闻与 H5 冲突，优先以 H5 为准。

### H4：高可信事实
官方政策、权威媒体新闻、行业权威数据、可验证市场数据、龙虎榜、正式披露的资金 / 交易数据。

适用说明：
- 通常可通过官方渠道或权威数据库交叉确认。
- 可作为板块催化和市场数据判断的重要依据。
- 若只看到转述而未见原始出处，应先下调处理。

### H3：正规信息
正规财经新闻、产业媒体报道、券商研报、机构纪要、平台涨停原因、异动解释。

适用说明：
- 可以提供有效信息增量，但常混有观点、推演与主观归因。
- 使用时必须拆分“事实段”与“分析段”。
- 不得直接替代公告或监管信息。

### H2：市场观点
游资复盘、盘前纪要、盘后复盘、自媒体分析、实盘赛文字、市场共识文章。

适用说明：
- 主要反映情绪、交易偏好、题材共识与市场解释框架。
- 可用于识别短线资金偏好，但不能作为公司层面事实依据。

### H1：弱线索
小作文、传闻、微信群截图、股吧 / 雪球传闻、未证实订单、未证实客户认证、未证实涨价。

适用说明：
- 默认不作为交易事实依据。
- 只可作为盘中线索、异动解释候选或资金攻击路径观察。
- 必须设置验证点，等待更高等级证据确认。

### H0：非证据
AI推理、模型总结、无来源搬运、无法确认来源的 OCR、纯猜测。

适用说明：
- 只能作为整理工具产物，不能进入事实判断链。
- 如需使用，必须回溯到原始信息源并重新评级。

## L：逻辑合理性

### L5：逻辑极强
直接对应公司核心业务，有明确订单、产品、客户、收入路径或公告支撑。

### L4：逻辑较强
与公司主营高度相关，产业链位置清晰，兑现路径较顺。

### L3：逻辑一般
题材相关，但需要验证收入占比、客户、订单、产能或财务影响。

### L2：逻辑偏弱
间接映射、参股映射、概念擦边、链条较长。

### L1：逻辑很弱
纯联想、远期想象、业务关联模糊。

### L0：逻辑冲突
与公告、主营、澄清、监管信息或已知事实明显冲突。

## C：环境适配度

### C5：高度适配
正好命中当前最强主线、核心题材、资金正在进攻的方向。

### C4：较强适配
命中活跃题材，已有板块或核心个股异动。

### C3：一般适配
题材有关注度，但不是当日最强方向。

### C2：弱适配
题材偏冷，需要等待资金验证。

### C1：低适配
市场暂时不认，扩散难度较大。

### C0：环境冲突
与当前风险偏好、资金方向或市场状态冲突，容易被兑现或无视。

## W：证伪窗口

### W5：长证伪窗口
短期极难验证或证伪，例如客户认证、供应链关系、海外大厂合作、远期订单、产业链卡位等短期难以确认的信息。叙事空间最大，容易被资金反复利用。

### W4：中长证伪窗口
几天内较难证伪，市场可以围绕该信息持续发酵，但仍需要等待公告、调研、新闻或产业数据验证。

### W3：中等证伪窗口
需要等待公告、互动易、调研、价格数据、订单数据、财报或行业数据验证。存在一定博弈空间。

### W2：短证伪窗口
1-2天内可能被核实、澄清或盘面否定。影响力有限，容易冲高兑现。

### W1：极短证伪窗口
盘中或次日就可能被打假、澄清或否定。高风险，只能谨慎观察。

### W0：窗口关闭
已经被公告、澄清、监管、权威媒体、董秘互动、客户 / 供应商信息、财报数据或盘面表现证伪。必须降级或排雷。

## I：影响力预期

### I5：引爆级
可能引发板块级扩散、核心股抢筹、盘中快速发酵。

### I4：强催化
可能推动核心个股或前排个股明显异动。

### I3：有效催化
有观察价值，可能影响局部资金选择。

### I2：弱催化
有题材关联，但扩散能力有限。

### I1：低影响
仅记录，暂不构成明显盘面影响。

### I0：无效或负影响
被澄清、证伪、监管、减持、业绩不及预期或兑现压制。

## 最终评级

### SSS
强事实 + 强逻辑 + 强环境 + 强资金共振，可能形成板块级主线催化。

### SS
逻辑强、环境强、资金有响应，具备强候选 / 强观察价值。

### S
有效催化，值得进入观察或候选池。

### A
逻辑成立，但需要进一步验证。

### B
有线索，但证据弱或环境一般。

### C
小作文、弱映射、纯情绪线索。

### R
降级、排雷、被证伪、与公告冲突、监管 / 减持 / 澄清压制。

## 使用口径建议
| 维度 | 代表含义 | 常见误区 |
|---|---|---|
| H | 真实性与可验证性 | 把 H 高等同于短线必涨 |
| L | 公司与题材映射是否顺 | 把擦边概念当核心受益 |
| C | 是否命中当前市场环境 | 忽略当日主线与风险偏好 |
| W | 短期内是否容易被证伪 | 把 W 高当成真实性高 |
| I | 盘面影响力预期 | 只看消息真假，不看资金是否会打 |
| 最终评级 | 短线博弈价值 | 把最终评级当成事实等级 |

## 升级与降级规则
- 同一信息若找到原始公告或监管链接，可提升 H 证据硬度。
- 只有截图、转述、摘要而无原文时，应降低 H 证据硬度。
- 与公告、监管文件冲突的新闻或传闻，H 不上调，最终评级直接降为 R。
- L0 逻辑冲突必须降级。
- C0 环境冲突应下调最终评级。
- W0 证伪窗口关闭，最终评级必须降为 R 或进入排雷。
- W1 / W2 且临近澄清、监管、问询、公告、业绩披露、互动易回复时，应降低最终评级。
- 高 W 只能放大影响力预期，不能替代 H 证据硬度。
- 不可证伪但逻辑牵强的信息，不得因为 W 高而升级。
- I0 负影响信息必须进入降级 / 排雷链路。
- 时效性失真的旧闻翻炒，可保留 H，但 C 与 I 应下降。

## 时间影响规则
- 公告还是公告，传闻还是传闻，时间不应改变 H。
- 盘前获取的信息，重点判断是否会影响开盘预期。
- 盘中获取的信息，重点判断是否与盘面异动同步。
- 盘后获取的信息，重点判断是否形成次日预期。
- 非交易日获取的信息，重点判断是否形成下一个交易日预期。
- 旧闻重抓时，H 可保持不变，但 C / I 应下调。
- 延迟获取时，应标记“可能已被市场消化”。
- 已部分兑现时，不得过度提升最终评级。
- 小作文盘中出现且板块同步异动，可提高 C / I，但不能提高 H。
- 临近公告、业绩披露、互动易回复、监管问询时，W 应下调。

## 来源升级限制
不得因为原文措辞强，就把弱来源升级为公告 / H5。

只有明确出现以下口径，或可回溯到正式披露文件时，才可以标为 H5：
- 公告
- 交易所披露
- 定期报告
- 业绩预告
- 监管文件
- 公司公告原文

以下表述通常不得直接升为 H5：
- 消息称
- 据悉
- 预计
- 传出
- 媒体称
- 业内人士表示
- 公司宣布
- 有望
- 或将
- 可能
- 计划
- 拟

处理建议：
- 原文只写“公司宣布”但未见正式披露，通常标 H3-H4。
- 原文只写“消息称 / 据悉 / 预计”，通常标 H2-H3。
- 社群、小作文、截图、无出处转述，通常标 H1。
- 无法确认来源时，标 H0-H1。
- 若后续核验到公告原文，可再升级为 H5。

## 输出要求
每条资讯进入“资讯裁判表”时，必须明确写出：
- H 证据硬度
- L 逻辑合理性
- C 环境适配度
- W 证伪窗口
- I 影响力预期
- 最终评级
- 给出验证点、证伪触发点与降级风险
