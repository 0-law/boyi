# 小作文处理规则

## 适用范围
适用于盘中流传文本、社群消息、传闻、微信群截图、朋友圈内容、论坛爆料、未证实产业链消息等。

## 核心定位
小作文不能当事实，但可以作为资金攻击线索。

## 基本原则
1. 默认低可信，H 证据硬度通常为 H1。
2. 只能作为“观察线索”，不能直接作为交易事实依据。
3. 必须追踪是否被更高等级证据确认或证伪。
4. 与公告、监管信息冲突时，必须立即降级。
5. 除 H 证据硬度外，必须单独评估 L 逻辑合理性、C 环境适配度、W 证伪窗口、I 影响力预期。
6. 小作文不能直接进入主攻池，但可以进入“观察”“小作文池”“高影响低可信观察”。
7. 不得因为小作文影响力高，就把它写成事实。

## 处理步骤

### 第一步：识别传闻类型
- 未证实订单
- 未证实客户认证
- 未证实涨价
- 未证实政策吹风
- 未证实并购重组
- 未证实业绩变化

### 第二步：记录传播特征
- 首次出现时间
- 传播渠道
- 是否多平台扩散
- 是否伴随股价异动
- 是否存在截图链条失真

### 第三步：设置验证点
每条小作文都必须设置至少一个验证点，例如：
- 是否有公告
- 是否有监管口径
- 是否有公司回应
- 是否有权威媒体跟进
- 是否有产业链多方交叉印证

## 使用边界
- 可用于识别题材发酵早期的资金攻击方向。
- 可用于解释盘中异常联动，但不能直接认定基本面变化。
- 若后续获得 H4/H5 支撑，可升级为正式催化链条的一部分。
- 若为 H1 但同时具备 L4/L5、C4/C5、W4/W5，且盘面已有明显资金异动，可标记为“高影响低可信观察”。
- 若 H1 + L2 以下，即使 W5，也不得升级为高评级，只能进入“小作文池”或“仅记录”。

## 典型降级场景
- 来源只剩二手截图
- 内容高度情绪化
- 金额、客户、时间都模糊
- 同一传闻多次变形传播
- 公司公告明确否认
- 董秘互动明确否认
- 权威媒体或客户供应商名单核查否认
- 业务不匹配或公司主营无法支撑
- 收入占比极低，难以支撑市场想象
- 仅送样、仅预研、未产品化、短期无收入利润
- 遇到监管提示或风险揭示
- 板块高开低走、核心股放量滞涨、后排炸板、资金撤退

## 输出建议
- 默认进入“观察”或“小作文池”。
- 小作文必须同步给出验证点和证伪触发点。
- 若 H1 + L4/L5 + C4/C5 + W4/W5 + 盘面异动明显，可标记为“高影响低可信观察”。
- 若 H1 + L2 以下，即使 W5，也不得升级为高评级。
- 若被公司公告、澄清、监管、权威媒体、董秘互动或盘面表现否定，W 直接降为 W0，最终评级必须降为 R。
- 若被公告澄清、监管提示、业务不匹配、收入占比极低，最终评级必须降为 R。
- 若与高等级证据冲突，直接标记“降级”或“排雷”。
- 若盘中热度高但无法证实，可记录为“情绪扩散”而非“事实催化”。
