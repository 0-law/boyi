# 资讯裁判自检清单

## 使用要求

AI 每次输出“资讯裁判结果”前，必须逐条检查以下问题。只要有一项未通过，就不能直接输出最终结果。

---

## 强制自检项

1. 是否把公告和小作文混为一谈？
2. 是否把复盘观点当成公司事实？
3. 是否把平台涨停原因当成硬逻辑？
4. 是否遗漏公告、涨停、连板、热榜、小作文中出现的股票和主体？
5. 是否编造股票代码？
6. 是否编造公司业务？
7. 是否把 AI 推理当成证据？
8. 是否没有标注小作文风险？
9. 是否没有标注澄清 / 监管 / 减持风险？
10. 是否给出了买卖建议、目标价或仓位建议？
11. 是否对每条资讯都给出 H / L / C / W / I 与最终评级？
12. 是否对每个交易主体都给出验证点？
13. 是否对传闻类信息设置降级条件？
14. 是否把 H证据硬度 当成最终短线评级？
15. 是否低估了高影响小作文？
16. 是否把小作文当成事实？
17. 是否只判断真实性，没有判断逻辑合理性？
18. 是否只判断真实性，没有判断环境适配度？
19. 是否只判断真实性，没有判断影响力预期？
20. 是否把公告真实性等同于短线影响力？
21. 是否对传闻类信息设置了澄清 / 证伪后的 R 级降级路径？
22. 是否把 W证伪窗口 当成了真实性？
23. 是否因为信息不可证伪，就无条件提高评级？
24. 是否检查了不可证伪信息的 L逻辑合理性？
25. 是否检查了不可证伪信息的 C环境适配度？
26. 是否对小作文设置了证伪触发点？
27. 是否识别了公司澄清、监管、问询、董秘互动、权威核查等证伪来源？
28. 是否识别了盘面证伪，例如高开低走、核心股放量滞涨、板块炸板、资金撤退？
29. 是否在 W0 出现后及时降级为 R？
30. 是否把高影响小作文写成了硬事实？

---

## 时间与时效自检

31. 是否完整读取了 `source_time / source_time_precision / source_session / ingest_time / ingest_session / analysis_time / timezone / timeliness_status`？
32. 是否把 `source_session` 和 `ingest_session` 混为一谈？
33. 是否错误地用 `ingest_time` 覆盖了 `source_session`？
34. 是否把“7月2日消息”硬补成精确分钟时间？
35. 是否把“7月2日晚”错误写成白天时段？
36. 是否在没有盘面数据时写出了 `已兑现 / 已部分兑现 / 高开低走 / 放量滞涨 / 资金承接强`？
37. 是否在催化窗口已经过去时仍维持过高时效状态？
38. 是否在没有依据时编造盘前、盘中、盘后或夜间？

---

## 主体与个股展示自检

39. `对应个股` 字段中是否残留 Markdown 链接？
40. `对应个股` 字段中是否残留本地绝对路径？
41. `对应个股` 字段中是否存在空链接或占位链接？
42. 是否把非 A 股主体误写成 A 股股票？
43. 是否对明确 A 股股票名称补齐了唯一 6 位代码，并避免对非 A 股主体、歧义主体或无法确认主体错误补码？
44. 是否遗漏交易主体映射表中的主体？
45. 是否遗漏背景主体 / 证据主体表中的主体？
46. 未进入交易主体表的主体，是否说明了原因？

---

## 信息性质与评级约束自检

47. 是否把 `预计 / 有望 / 或将` 直接写成事实？
48. 是否把 `拟 / 计划` 直接写成已落地事件？
49. 是否把 `据悉 / 消息称 / 知情人士 / 供应商` 直接写成公告事实？
50. 是否把 `网传 / 爆料 / 小作文 / 大V` 直接写成硬事实？
51. 是否让海外主体、未上市主体、海外政策或产业新闻在没有 A 股订单、公告、业绩或盘面验证前直接达到 `SS`？
52. 是否因为 W 很高就忽略了 L 很弱的问题？
53. 是否把“资讯侧主攻支撑”误写成了交易主攻结论？

---

## 逻辑疑点与回放自检

54. 是否对存在逻辑跳跃、时间矛盾、因果偷换的内容标注了 `原文逻辑疑点`？
55. `疑点说明` 是否写清楚了问题，而不是只写“有风险”？
56. 是否把所有独立催化拆成了独立行？
57. 是否保留了足够的验证点与证伪触发点，便于后续回放与复核？

---

## 最小通过标准

- 每条资讯都有统一时间字段组。
- 每条资讯都有 H / L / C / W / I 与最终评级。
- 每条资讯都有验证点、证伪触发点、原文逻辑疑点与处理结果。
- 交易主体与背景主体已经拆表。
- 无 Markdown 链接污染、无本地路径污染、无空链接污染。
- 无买卖建议、目标价、仓位建议。
