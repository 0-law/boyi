# 资讯裁判样例输出

## 样例输入
以下为虚构示例，仅用于展示裁判方法：

1. 2026-07-03 08:20，甲公司发布公告，称中标某省级电网设备采购项目，金额明确，公告已披露。
2. 2026-07-03 08:45，权威媒体发布新闻，称某部委发布新型电网升级支持政策，原文可回溯。
3. 2026-07-03 09:05，某财经平台快讯提到乙公司可能受益于电网升级，但未引用公司公告。
4. 2026-07-03 09:18，社群流传小作文，称丙公司“已经拿下核心客户大单”，未见原始来源。
5. 2026-07-03 10:02，某复盘文章称“市场认为乙公司会成为电网线核心弹性标的”，属于盘后观点总结。
6. 2026-07-03 10:30，甲公司发布澄清公告，否认市场流传的“新增超大额海外订单”传闻。

## 样例说明
- 样例一：硬公告，体现 H5 + L5 + C4/C5 + W 不作为主要加分项 + I4/I5，可评 SS / SSS。
- 样例二：高影响小作文，体现 H1 + L4 + C5 + W5 + I4，可评 S / SS 观察，但不能写成事实。
- 样例三：空泛小作文，体现 H1 + L1/L2 + C3 + W5 + I1/I2，不得因为不可证伪而升级。
- 样例四：澄清公告关闭证伪窗口，体现 H5 澄清 + 原小作文 W0 + L0 + I0，最终评级 R。
- 样例五：盘面证伪，体现 W 从 W4 下调到 W2 / W1，I 从 I4 下调到 I1 / I2。

## 样例结论提示
本样例展示了三类最常见处理方式：
- 硬公告可以给到 H5，但主要还是靠 L、C、I 决定是否进入 SS / SSS。
- 高影响小作文可以是 H1 + W5，但只能作为高影响低可信观察，不能写成事实。
- 不可证伪但逻辑牵强的信息，不能因为 W 高而升级。
- 澄清公告会直接把原传闻的 W 打到 W0，并触发 R。
- 即使没有公告证伪，盘面高开低走、核心股放量滞涨、后排炸板，也属于盘面证伪，需要下调 W、I 和最终评级。
